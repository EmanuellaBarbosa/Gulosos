# Gulosos
Pasta criada para os algoritmos PRIM e Dijkistra

## Para usar o Algoritmo de PRIM:
#### Comenta-se as linhas de Dijkistra
#### Dá o make
#### ./algulosos < ainstancia.txt

## Para usar o Algoritmo de Dijkistra:
#### Comenta-se as linhas de PRIM
#### Dá o make
#### ./algulosos < ainstancia.txt
